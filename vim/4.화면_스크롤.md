# 4.화면\_스크롤

- `ctrl + e` : 스크롤을 한줄 아래로 이동
- `ctrl + y` : 스크롤을 한줄 위로 이동
- `ctrl + u` : 커서를 반페이지씩 위로 이동
- `ctrl + d` : 커서를 반페이지씩 아래로 이동
- `gg` : 문서의 처음으로 이동
- `shift + g` : 문서의 마지막으로 이동
- `:5` : 5번째 행으로 이동
- `5gg` : 5번째 행으로 이동
