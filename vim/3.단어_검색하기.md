# 3.단어 검색하기

- `/` : 커서 이후로, 찾고 싶은 문자열을 입력 후 `Enter`
- `?` : 커서 이전의, 찾고 싶은 문자열을 입력 후 `Enter`
- `/^` : 시작하는 문자열 검색
- `n` : 다음에 찾고 싶은 문자열로 이동
- `N` : 이전에 찾고 싶은 문자열로 이동
- `/asdf` : 검색 하이라이트 끄기

  > 비정식적인 방법

- `*` : `/` + 커서가 존재하는 특정 문자열을
