### vim mode
1. normal mode
2. insert mode
3. command mode
4. visual mode

### normal mode
1. 커서를 움직인다
2. 명령을 수행한다

### insert mode
1. i or a 로 해당 모드 진입
2. esc 클릭 시 normal 모드로 변경
3. 기능
  - 타이핑
  - 커서 이동
  - ctrl 키와 조합하여 일련의 명령어 수행

### command mode
1. : 을 통해 해당 모드 진입
2. 기능
   - 문서의 저장
   - 프로그램(또는 탭) 종료
   - 문자열 찾기, 바꾸기
   
### visual mode
1. ctrl-v 또는 shift-v를 통해 해당 모드 진입
2. 마우스 드래그처럼 블럭을 씌우는 역할
3. 문서 내 범위 지정
