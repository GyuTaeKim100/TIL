# 모드

## 일반 모드

- 커서 움직임

## 입력 모드

### 입력

- `a`: 커서 다음에 입력 커서 활성화
  > append
- `i`: 커서 이전에 입력 커서 활성화
  > insert
- `o`: 커서 아래로 한줄이 생기고 입력 커서 활성화
  > open line
- `r`: 대체, 입력 후 바로 일반모드로 전환
  > replace
- `A`: 행 마지막에 입력 커서 활성화
- `I`: 행 처음에 입력 커서 활성화
- `O`: `o` + 방향 위
- `ctrl + [` : `esc`와 동일

### 지우기

> 지우기는 잘라내기로 동작함

- `ctrl - h` : `backspace`와 동일
- `x` : 문자 지우기
- `dd` : 한 문장을 지움
- `D` : 커서 다음 모든 문자열을 지움
- `J` : 밑에 줄을 위에 줄로 당겨옴

## 복사

- `p` : 잘라내기 또는 복사한 내용을 아래에 복사
- `P` : 잘라내기 또는 복사한 내용을 위에 복사
- `yy`: 한 문장 복사

## 명령형 모드

- `:` : 진입
- `esc` : 종료
- 예시 `:q` 등

## 비주얼 모드

> block을 선택, 복사, 삭제 가능

- `v` : 일반 비쥬얼 모드
- `x` : 삭제
- `y` : 복사
- `V` : 행 단위 비쥬얼 모드
- `ctrl + v` : 비쥬얼 블록 모드
- `gg + v + shift g` : 전체 선택
