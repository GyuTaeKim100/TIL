    - 보충
        - 데이터 전송 시 시간이나 공간을 분할하는 기술이 사용되는데, 시분할이란 개별적인 비트들이 (한개의 회선을 통해서) 차례대로 보내지는 것을 말하고 공간분할은 여러 개의 회선이나 통로를 통해 여러 개의 비트들이 병렬로 보내어질 수 있다는 것을 말한다.
        - 컴퓨터 하드웨어나 데이터 전송 측면에서 직렬접속, 직렬운영 그리고 직렬매체는 보통 단순하고 느린 운영을 가리킨다. (예시로 직렬 포트에 접속되어 있는 마우스, 실제로 작성하면서 아무것도 떠오르지 않는 중이다)
        - 병렬접속이나 운영은 많은 양의 데이터가 프린터로 보내지는 것과 같이 빠른 운영을 가리킨다.
        - 단 직렬전송을 하는 매체 중에 광케이블과 같은 매체는 동시에 여러 개의 신호로 병렬로 보내는 매체보다 훨씬 더 빠르게 데이터를 전송하므로, 결과적으로 직렬전송이 병렬전송보다 느리다는 것이 항상 옳은 얘기는 아니다.
        - 전통적인 컴퓨터 또는 프로그램들은 직렬 형식으로 운영된다고 볼 수 있다. 이유는 그 컴퓨터들은 프로그램을 읽어서 한 번에 한걔씩 명령어를 차례로 수행하기 때문이다.
        - 오늘날의 컴퓨터는 명령어를 분할한 다음, 이를 여러 개의 프로세서들을 이용하여 병렬로 처리하는 것도 있다. (이경우 시분할로 구현된 병렬이다)

## 클립 보드
- 클립보드(clipboard, ‘종이 끼우개’라는 뜻)는 잘라내서 붙여넣는 과정을 통해 문서나 응용 프로그램 사이에 자료가 전송될 때 짧은 시간 동안 해당 자료를 저장하는 데에 사용되는 소프트웨어 프로그램이다. 보통 그래픽 사용자 인터페이스 환경의 일부이며 환경 안에 있는 대부분의 프로그램으로부터 접근할 수 있는 임시 메모리 블록과 동의어이다.
- 예시 (Mac 기준)
    - Cmd + C: 자료를 복사하여 클립보드에 저장
    - Cmd + X: 자료를 잘라내어 클립보드에 저장
    - Cmd + V: 클립보드에 저장된 자료를 붙여넣기
